# 🧤 월드컵 핸드 골키퍼 · 2026 A조

웹캠으로 **양손**을 움직여 골키퍼 장갑으로 슛을 막는 게임입니다.
2026 월드컵 **A조**(🇰🇷대한민국·🇲🇽멕시코·🇨🇿체코·🇿🇦남아공) 시나리오에서
원하는 국가의 **실제 주전 골키퍼**를 골라 조별리그 3경기를 치르고 **32강 진출국**을 가립니다.

> ※ 내 모습은 화면에 나오지 않습니다. 웹캠은 **손 추적 전용**으로만 쓰여요.

## ▶ 실행 방법

> ⚠️ 카메라(getUserMedia)는 보안 컨텍스트가 필요해서 `index.html`을
> **더블클릭(file://)** 하면 작동하지 않습니다. 반드시 **localhost** 로 실행하세요.

### 가장 쉬운 방법
1. `start.bat` 을 **더블클릭** → 서버 실행 + 브라우저 자동 열림
2. **카메라 시작 ▶** → 카메라 권한 **허용**
3. 끝낼 땐 검은 서버 창에서 `Ctrl + C`

### 수동 실행
```bash
cd Move_Game
python -m http.server 8000
# http://localhost:8000/index.html 접속
```

## 🎮 게임 흐름
1. **국가 선택** — A조 4개국 중 플레이할 나라
2. **골키퍼 선택** — 그 나라의 실제 주전급 GK 3인 (사진·이름·나이·소속팀)
3. **조별리그 3경기** — 상대 공격수가 **슈팅 또는 헤딩**, 날아오는 공을 양손으로 막기 (경기당 슛 5개)
4. **경기 결과** — 내 경기 + 같은 차전 다른 경기 결과 + 실시간 순위표
5. **최종 결과** — 조 1·2위가 **32강 진출**

- 슛을 막으면 **SAVE**, 못 막으면 **실점**. 실점이 적을수록 승리 확률 ↑
- 내 팀 득점은 전력차와 선방 수에 따라 산출됩니다 (잘 막을수록 유리).
- 다른 조 경기는 각 팀 전력 기반으로 자동 시뮬레이션됩니다.

## 🧤 골키퍼 데이터 출처
각국 **주전급 골키퍼 3인**의 이름·나이·소속팀은 **Wikipedia / FIFA / 각국 협회** 등
사실 확인된 출처에서 2025-26 시즌 기준으로 조사했습니다. 사진은 Wikimedia Commons
자유이용 이미지를 사용하며(자유 이미지가 없는 일부 백업 GK는 이니셜 아바타로 대체),
주요 변동도 반영했습니다(예: 멕시코 말라곤 아킬레스건 부상 → 라울 랑헬 주전).

> A조 4개국 구성은 **사용자가 지정한 시나리오**이며, 실제 FIFA 공식 조 추첨과 다를 수 있습니다.

## 💡 잘 안 될 때
- **카메라가 안 켜져요** → 주소가 `http://localhost:8000` 인지 확인 (file:// 불가)
- **손이 인식 안 돼요** → 양손바닥이 카메라를 향하게, 밝은 곳에서 플레이
- **사진이 안 보여요** → 인터넷 연결 확인 (Wikimedia/CDN 로딩 필요). 안 뜨면 이니셜로 대체됩니다
